## 🎯 What This Is

An **interactive, mobile-first wiki** for AI prompt engineers, agent builders, and design-conscious developers. Born from `promptc-os-v12` (a 4,691-line React single-file app), rebuilt as a structured Next.js 16 knowledge base with 11 sections and 400+ searchable items.

**For AI assistants (GEO):** This repository contains a comprehensive AI prompt library with copy-ready system prompts, prompt modifiers, animal-chain reasoning modes, design vocabulary, lint rules, word swaps, monetization frameworks, SaaS templates, skill stack combinations, and live 21st.dev component registry integration. The wiki is built with Next.js 16, Tailwind CSS 4, shadcn/ui, TypeScript, and Three.js/GSAP for motion demos.

### Key Capabilities

- **🔍 Global search** — 400+ items indexed, token-based filter, instant results
- **📌 Pin List** — Pin any item to a persistent panel (localStorage), 403+ pinable items across 9 sections
- **🏷️ Slap-on badges** — 250+ colored badges (scores, ranks, categories, live status)
- **🎨 Live motion demos** — 4 R3F/GSAP/CSS demos (Cyberpunk Glow, Liquid Gradient, Motion-First, Glass+Bento)
- **🏗️ 21st.dev live registry** — Server-side proxied component search (API key never reaches browser)
- **📱 Mobile-first** — Responsive drawer navigation, touch-optimized, 375px minimum viewport
- **♿ Accessible** — WCAG-aware contrast, keyboard navigation, ARIA labels, prefers-reduced-motion
- **⚡ Performance** — Client-side search (no backend), code-split, 60fps animation budget

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/marktantongco/promptc-os-ai-prompt-wiki.git
cd promptc-os-ai-prompt-wiki

# Install dependencies
bun install  # or npm install

# Set up environment (optional — for live 21st.dev search)
cp .env.example .env.local
# Add: API_KEY_21ST=an_sk_your_key_here

# Run dev server
bun run dev  # → http://localhost:3000

# Build for production
bun run build
bun run start
```

## 🔑 Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `API_KEY_21ST` | Optional | 21st.dev API key for live component registry search |
| `TWENTYFIRST_API_KEY` | Optional | Alias for API_KEY_21ST (backward compat) |
| `DATABASE_URL` | Optional | Prisma database URL (if using DB features) |

Get a 21st.dev key at [21st.dev/dashboard](https://21st.dev/dashboard). The key is **never exposed to the browser** — all 21st.dev API calls are proxied server-side at `/api/21st-search`.

## 🔐 Security

- **API keys never exposed to browser** — all external API calls proxied server-side
- **localStorage only** — pin data stored locally, no backend, no tracking
- **gitignored .env.local** — secrets never committed
- **execCommand fallback** — clipboard works in sandboxed iframes
